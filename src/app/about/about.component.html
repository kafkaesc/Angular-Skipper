About This Project

Built by Jared Hettinger

This project is built in Angular following HTML5 and CSS3 best practices. The project utilizes TypeScript for static typing and reliable refactoring. Bootstrap is installed to provide a responsive UI. Not included in the site itself but still incredibly helpful is the Angular CLI to generate and manage files.

If you are using Safari or Firefox on macOS and this site is not behaving as expected, you might want to check your system settings. You can find more details about this on Stack Overflow .
