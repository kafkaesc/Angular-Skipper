My Favorite Things

A practice on accessibility

Welcome to My Favorite Things! This website is a demo for how to effectively implement a skip to main content link in Angular.

If you tab through the page using only your keyboard you'll notice a hidden button will appear after the MFT logo. Hitting enter will nake your cursor skip to the main content of whichever page you're on.

There is also a secondary navigation menu demostrating different possible methods for implementing the link. You can open the menu by hitting enter on 'Show Skip Demos.' If you want to return to the beginning of that menu to try other links, you can hit enter on the back link at the end of the menu.
